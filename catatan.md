# ReactJs

## install reactJs

```
npx create-react-app my-app
cd my-app
npm start

```

### how to use react js
the first time use react js you must know about javascript basic

### using state 
to use a state from react you must install axios 
```
yarn add axios
npm install axios

```
and then import to your project 

```
import axios from 'axios'
```
